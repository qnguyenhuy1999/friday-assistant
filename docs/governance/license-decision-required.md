# License Decision Required

This repository does not yet have an explicit license selection from the
repository owner.

**Public distribution, external contribution, and downstream reuse of this
repository are blocked until a license is explicitly chosen.**

No license status from any other repository (including Javis OS) has been
carried over. This decision must be made independently for Friday Agent OS.

## What needs to happen

1. The repository owner selects a license (e.g. MIT, Apache-2.0, proprietary,
   or another option).
2. A `LICENSE` file is added at the repository root reflecting that choice.
3. This file is removed once `LICENSE` exists.
4. Any license-sensitive documentation (README, CONTRIBUTING) is updated to
   reference the chosen license.

## Why this file exists instead of a guess

Silently defaulting to a license would encode a legal decision without owner
sign-off. Per repository governance rules, license selection is not something
tooling or an assistant may infer or assume.

See [provenance.md](provenance.md) for related rules on third-party code reuse
and licensing compatibility.
